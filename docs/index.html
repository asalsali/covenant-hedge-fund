How it works

From market data to
portfolio decisions

Three stages. Fully deterministic. Same inputs always produce the same trades.

1

Data Collection

Market prices, financial metrics, and insider activity from Yahoo Finance and CoinGecko. Free data sources. Supports equities and crypto.

2

Multi-Agent Analysis

18 analyst personas each produce a scored signal with direction, confidence, and reasoning. Quant analysts are fully deterministic. Value and macro use open-source models for qualitative reasoning.

3

Risk-Managed Portfolio

Signals combined via confidence-weighted quorum voting. The risk engine enforces volatility limits, correlation constraints, and position caps as code.

Architecture

Governed pipeline,
not a chatbot wrapper

Every stage is deterministic and auditable. The LLM is one input into a governed system — not the system itself.

  • Deterministic by default. The 5 quant analysts run without any LLM. Same inputs always produce the same signals, votes, and trades.
  • Open-source models, not paid APIs. Value and macro analysts use Llama 3, Qwen 2.5, DeepSeek, or Mistral via Ollama. Free, local, private.
  • Risk as law. 25 risk rules enforced in code: position sizing caps, volatility limits, correlation constraints. No model can override them.
  • Full audit trail. Every signal, every vote, every allocation decision is logged. Trace any trade back to the analysts that recommended it.
