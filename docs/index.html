Methodology

How it works

A governed pipeline from market data to trade execution. Each stage is deterministic and auditable.

1

Data Collection

Market prices, financial metrics, and insider activity are fetched from Yahoo Finance for the target universe. No paid data feeds required.

2

Multi-Agent Analysis

18 analyst personas -- quant, value, and macro -- each produce a scored signal with confidence and reasoning. Quant analysts run without any LLM. Value and macro analysts use optional LLM augmentation and degrade gracefully without one.

3

Risk-Managed Portfolio

Signals are combined via confidence-weighted scoring. The risk engine enforces volatility limits, correlation constraints, and position caps in code. Same inputs always produce the same trades.

Zero Configuration

Run the demo in one command

No API keys. No configuration files. Clone the repo, install dependencies, and run. You get a full 6-month backtest on AAPL, MSFT, and NVDA with an interactive HTML report.

Differentiation

Governance over guesswork

Most AI hedge fund projects treat LLMs as the decision-maker. This one treats them as one input into a governed pipeline.

Covenant Hedge Fund Typical AI Hedge Fund
LLM required to trade No (quant-only mode) Yes
Decision determinism Same inputs = same outputs LLM variance on every run
Risk enforcement 25 code-enforced rules LLM-suggested limits
Cost to run $0 (quant-only or Ollama) Paid API keys required
